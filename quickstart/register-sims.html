Register emnify SIMs

Once you get your emnify SIMs, you need to register them before you can start using them. If you order the free Evaluation SIM package, you'll have to register each of the three SIMs separately.

  1. Log in to your emnify account.

  2. From the Get Started menu, select Register SIM(s)

  3. Since the Single-SIM registration is already selected by default, enter the Batch Identification Code (BIC1) in the prompt:

  4. If you ordered more than five SIM cards, select SIM Batch.

  5. Enter the BIC2 number for the SIM batch.

Create a new device​

As part of the SIM registration, you can create a new device for the newly registered SIM while you are still in the SIM registration dialog.

Show an example
  1. After confirming your BIC1 is correct, select Register SIM Card.

  1. After your SIM is registered, select Create Device.

  2. In the Create a new device dialog, make sure you replace Add Device name with an appropriate name for the new device.

  3. Inspect your choices for device name and tags before selecting Create Device.

  4. Confirm your choice to create and activate the device by selecting Activate.

  5. Once created, your device should be ready to go online if it has been configured with the correct APN and data roaming is enabled.

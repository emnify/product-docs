1. [Log in to your emnify account](https://portal.emnify.com/sign).
2. Navigate to **Integrations** and find the [**Secure Connection**](https://portal.emnify.com/integrations#secure-connection) section.
3. Under **OpenVPN**, select **Show instructions**.

<img
  src={require('../assets/portal-integrations-secure-connection-openvpn-show-instructions.png').default}
  alt=""
/>

4. Make sure you're on the **{props.operatingSystem}** tab, then click **Download {props.operatingSystem == 'Linux' ? 'client.conf' : 'client.ovpn'}**.

<img
  src={require(`../assets/portal-integrations-secure-connections-openvpn-${props.imageSrc}-download.png`).default}
  alt=""
/>

Select the desired region or choose **Download all regions**.

<img
  src={require(`../assets/portal-integrations-secure-connections-openvpn-${props.imageSrc}-download-dropdown.png`).default}
  alt=""
  style={{ height: 400 }}
/>

Once downloaded, store that file on your server in the folder <code>{props.operatingSystem == 'Windows' ? '\\OpenVPN\\config' : '/etc/openvpn'}</code>.

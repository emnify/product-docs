Select the filter you want to edit and enter a value. 
To add multiple filter values, press the `Tab` key.
Once you've entered all the values you want to filter by, press the `Enter` key or click **Apply**.

To remove a filter or value, click the corresponding **X**.

Filters can be combined or used together with the search bar.
Filtered values are saved in the URL as a query parameter (for example, after filtering for {props.tableValueType} named `Alexa`, the URL includes `&name=Alexa` at the end).
